# 🎨 Grid Layout Dashboard

A modern and responsive **Grid Layout Dashboard** built using **HTML5** and **CSS3**. This project demonstrates advanced CSS Grid concepts, card-based UI design, responsive layouts, typography, and modern dashboard styling.

## 📱 Responsive Design

The dashboard automatically switches to a **single-column layout** on devices smaller than **768px**.

```css
@media (max-width:768px){
    main{
        grid-template-columns:1fr;
    }
}
```
